Onboarding & activation (required)

These steps are mandatory before bot activation on live MT5 accounts.

  1. MT5 setup: Tools → Options → Expert Advisors → enable Allow WebRequest for listed URL.
  2. Add URL exactly: https://mt5-license-server-production.up.railway.app.
  3. Submit: Bot license key, MT5 account number, and checkout email.
  4. Attach EA: enable Algo Trading and confirm no WebRequest/license errors in Experts/Journal.

If MT5 account number is missing at purchase, activation stays pending until provided.
