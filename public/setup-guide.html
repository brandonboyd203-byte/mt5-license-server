Setup Guide

Use this quick path for broker setup, MT5 installation, and first login.

1. Broker + MT5

  • Create broker account and MT5 demo/live account.
  • Install MT5 on VPS or local terminal.
  • Confirm server and account login are stable.

2. Bot Install

  • Attach EA to correct symbol/timeframe.
  • Enable Algo Trading and DLL permissions if required.
  • Validate license/account activation.

3. Monitoring

  • Confirm telemetry feed updates in motherboard.
  • Set daily reset and reporting windows.
  • Wire Discord/Telegram alerts for failures.
